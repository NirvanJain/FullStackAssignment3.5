# 🚀 Frontend Foundation - Creator Platform

This project is a **React-based frontend foundation** for a Creator Platform. It focuses on building a clean project structure, implementing client-side routing, and setting up reusable components for future development.

---

## 📌 Features

* ⚛️ Built using **React + Vite**
* 🔗 **Client-side routing** with React Router v6
* 🧩 Reusable components:

  * Header
  * Footer
* 📄 Multiple pages:

  * Home
  * Login
  * Register
  * Dashboard
  * 404 (Not Found)
* 🚫 No full page reloads (SPA behavior)
* 📁 Clean and scalable folder structure

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/NirvanJain/FullStackAssignment3.5.git
```

2. Navigate to the project:

```
cd client
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```

5. Open in browser:

```
http://localhost:5173
```

## 🔗 Routing Overview

| Route        | Page          |
| ------------ | ------------- |
| `/`          | Home          |
| `/login`     | Login         |
| `/register`  | Register      |
| `/dashboard` | Dashboard     |
| `*`          | 404 Not Found |
